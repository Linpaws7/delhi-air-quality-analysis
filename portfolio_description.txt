Built an end-to-end air-quality forecasting pipeline for Delhi using open-source air-pollution and meteorological data. Collected, cleaned, merged, and transformed daily pollutant and weather variables to predict next-day PM2.5 and generate pollution-risk categories. Compared baseline, Linear Regression, and Random Forest models, and found that the baseline persistence model outperformed the tested machine learning approaches. Produced model comparison tables, risk summaries, correlation outputs, and visualisations to support both portfolio presentation and research-paper development.
